# Set-PowershaiActiveChat

## SYNOPSIS <!--!= @#Synop !-->
Ändert den aktiven Chat

## DESCRIPTION <!--!= @#Desc !-->
Der aktive Chat ist der Standard-Chat, in den Powershai Chat-Befehle gesendet werden.  
Wenn nur ein Chat vorhanden ist, wird dieser standardmäßig als aktiv betrachtet.

## SYNTAX <!--!= @#Syntax !-->

```
Set-PowershaiActiveChat [[-ChatId] <Object>] [<CommonParameters>]
```
