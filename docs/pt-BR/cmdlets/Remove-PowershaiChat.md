# Remove-PowershaiChat

## SYNOPSIS <!--!= @#Synop !-->
Remove um chat e retorna o objeto com todo o histórico e contexto

## SYNTAX <!--!= @#Syntax !-->

```
Remove-PowershaiChat [[-ChatId] <Object>] [<CommonParameters>]
```

## PARAMETERS <!--!= @#Params !-->

### -ChatId
Id do chat a ser removido, ou objeto representando o chat!
