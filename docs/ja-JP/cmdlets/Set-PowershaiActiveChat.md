# Set-PowershaiActiveChat

## SYNOPSIS <!--!= @#Synop !-->
アクティブなチャットを変更します

## DESCRIPTION <!--!= @#Desc !-->
アクティブなチャットは、Powershai チャットのコマンドが送信されるデフォルトのチャットです。  
チャットが 1 つしかない場合、デフォルトでアクティブと見なされます。

## SYNTAX <!--!= @#Syntax !-->

```
Set-PowershaiActiveChat [[-ChatId] <Object>] [<CommonParameters>]
```
