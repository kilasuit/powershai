# Set-PowershaiActiveChat

## SYNOPSIS <!--!= @#Synop !-->
Modifie le chat actif

## DESCRIPTION <!--!= @#Desc !-->
Le chat actif est le chat par défaut vers lequel les commandes Powershai Chat sont envoyées.  
Si un seul chat existe, il est considéré comme actif par défaut.

## SYNTAX <!--!= @#Syntax !-->

```

```
