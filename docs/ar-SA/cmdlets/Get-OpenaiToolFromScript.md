# Get-OpenaiToolFromScript

## SYNOPSIS <!--!= @#Synop !-->


## DESCRIPTION <!--!= @#Desc !-->
دالة مساعدة لتحويل ملف نصي .ps1 إلى تنسيق مخطط متوقع من قبل OpenAI.

	tools - قائمة بالأدوات، ليتم إرسالها في استدعاء OpenAI.
