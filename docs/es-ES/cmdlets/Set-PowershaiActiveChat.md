# Set-PowershaiActiveChat

## SYNOPSIS <!--!= @#Synop !-->
Cambia el chat activo

## DESCRIPTION <!--!= @#Desc !-->
El chat activo es el chat predeterminado al que se envían los comandos de Powershai Chat.  
Si solo hay 1 chat, se considera activo de forma predeterminada.

## SYNTAX <!--!= @#Syntax !-->

